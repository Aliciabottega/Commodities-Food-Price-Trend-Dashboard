  ## Project Overview
  This project analyzes food price trends over time to identify patterns, price volatility, and inflationary movements across food categories.
  The goal is to generate insights that support better decision-making by understanding how food prices change across periods.

  ## Data Source
  This data source is a "Nigerian Commodity Price and market.csv" file.
  
  ## Tools
     - Cleaning : Excel ( power query )
  
     - Dashboard : Excel

  ## Data Cleaning
  The dataset required several cleaning steps to ensure accuracy and usability:

   - Standardized date formats for time-series analysis

   - Ensured price columns were stored in numeric / customized formats

   - Filtered out incomplete periods to avoid misleading trend calculations

   - Corrected inconsistent text formatting by standardizing capitalization across all categorical fields
